Privacy Policy – HoloScan

Effective Date: May 29, 2025

1. General Information

HoloScan is a mobile application designed to help users catalogue and manage their personal collection of Star Wars Unlimited trading cards. The app can be used fully offline without creating an account. Users may optionally sign in to enable Premium features, including cloud synchronization.

2. Data Collected

  • Offline use (no login / no subscription): Card collection data entered manually by the user (card names, quantities, conditions, notes). This data is stored only on the user's device and is never transmitted externally.
  • Login without Premium subscription: Authentication data from Google or Apple Sign-In handled by Firebase Authentication (e.g., email address, display name, unique user ID). Subscription status metadata handled by RevenueCat. Card collection data remains local only and is not uploaded or synchronized.
  • Login with Premium subscription (required for cloud sync): Authentication data (Firebase Authentication) and subscription data (RevenueCat). The user's card collection, decks, and lists are stored in Firebase Cloud Firestore for cloud backup and synchronization.
  • Technical data: Locally cached preferences (e.g., subscription status) stored using SharedPreferences on the device.

3. Purpose of Data Use

  • To provide basic functionality for local collection management on device.
  • To enable Premium-only features, including cloud backup and synchronization via Firebase Cloud Firestore.
  • To manage Premium subscriptions and purchases via RevenueCat (validation, restoration across devices).

4. Data Storage

  • Local Storage: By default, all collection data is stored only on the user's device.
  • Cloud Storage (Premium users only): When subscribed and logged in, user data (collections, decks, lists) is stored in Google Firebase Cloud Firestore and associated with the user's account.
  • Subscriptions: Subscription and purchase information is processed and stored by RevenueCat.
  • The developer does not access individual user data directly.

5. Data Sharing

The app does not sell or share data with third parties for advertising or marketing purposes. The following service providers are used strictly to deliver app functionality:

  • Google Firebase: Authentication (Google/Apple Sign-In via Firebase Authentication) and, for Premium users only, cloud storage (Cloud Firestore).
    View Google Privacy Policy
  • Apple (Sign in with Apple): Authentication for iOS/macOS users.
    View Apple Privacy Policy
  • RevenueCat: Subscription and purchase management.
    View RevenueCat Privacy Policy

6. Tracking Technologies

The app does not use advertising SDKs, cookies, or analytics tools. Firebase Authentication and RevenueCat may process device and account identifiers strictly for login, fraud prevention, and subscription validation.

7. Data Retention and Deletion

  • Offline/local users: Data exists only on the device and can be deleted by uninstalling the app or using in-app options to clear data.
  • Premium users: Cloud data stored in Firebase can be deleted upon request by contacting the developer (see Section 9).
  • Subscription data retention follows App Store and Google Play billing policies (managed by RevenueCat).

8. Children's Privacy

The app is not intended for use by children under the age of 13. If you believe a child has used the app and provided personal data, please contact the developer for deletion.
